Bdd Videoannotator

Why ?

BDD-based tests should improve the way you integrate the stakeholders in the development process. But is a textually story really suitable to describe the handling of the software using the GUI?

You can already set up click tests using automation Frameworks like Selenium to execute such an acceptance test. But as a result you only get a textual logfile that describes what happened on the front end.

To close this gap BDD Videoannotator can record a screencast that contains the steps specified in the BDD-Feature-Files.

Usage

BDD Videoannotator is a plugin that can be used for various BDD-Tools. Currently there are two sample projects showing the usage of BDD-Videoannotator:

  • Cucumber JVM in Java
  • Behat in PHP
